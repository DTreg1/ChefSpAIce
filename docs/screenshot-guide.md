# ChefSpAIce Screenshot Guide for App Stores

## Required Screenshots

### Apple App Store

**iPhone 6.7" Display (Required)**
- Device: iPhone 14 Pro Max, iPhone 15 Pro Max
- Resolution: 1290 x 2796 pixels
- Minimum: 3 screenshots, Maximum: 10

**iPhone 6.5" Display (Required)**
- Device: iPhone 11 Pro Max, iPhone XS Max
- Resolution: 1242 x 2688 pixels
- Minimum: 3 screenshots, Maximum: 10

**iPhone 5.5" Display (Required for older support)**
- Device: iPhone 8 Plus, iPhone 7 Plus
- Resolution: 1242 x 2208 pixels
- Minimum: 3 screenshots, Maximum: 10

**iPad 12.9" Display (if supporting iPad)**
- Resolution: 2048 x 2732 pixels

---

### Google Play Store

**Phone Screenshots (Required)**
- Minimum: 2, Maximum: 8
- Resolution: 16:9 or 9:16 aspect ratio
- Recommended: 1080 x 1920 pixels (minimum 320px, max 3840px per side)

**Tablet 7" Screenshots (Optional but recommended)**
- Resolution: 1080 x 1920 pixels

**Tablet 10" Screenshots (Optional)**
- Resolution: 1920 x 1200 pixels

**Feature Graphic (Required)**
- Resolution: 1024 x 500 pixels
- Used for store listing header

## Recommended Screenshot Sequence

Create screenshots showing these key features in order:

### Screenshot 1: Welcome/Hero
**Caption:** "Your AI-Powered Kitchen Companion"
- Show the main inventory screen with a few food items
- Clean, organized view demonstrating the app's purpose

### Screenshot 2: Inventory Tracking
**Caption:** "Track Every Item in Your Kitchen"
- Show inventory with items organized by storage location
- Include expiration dates visible
- Show different food categories

### Screenshot 3: AI Recipe Generation
**Caption:** "Get Recipes From What You Have"
- Show the AI chat or recipe generation screen
- Display a generated recipe based on inventory
- Highlight the AI-powered badge

### Screenshot 4: Expiration Alerts
**Caption:** "Never Let Food Go to Waste"
- Show items with expiration warnings
- Display notification or alert indicators
- Emphasize the waste-reduction feature

### Screenshot 5: Meal Planning
**Caption:** "Plan Your Week Effortlessly"
- Show the meal planning calendar
- Display meals assigned to days
- Show integration with inventory

### Screenshot 6: Barcode Scanning
**Caption:** "Add Items in Seconds"
- Show the barcode scanner interface
- Display scanned item being added
- Emphasize convenience

### Screenshot 7: Shopping Lists
**Caption:** "Smart Shopping Lists"
- Show a shopping list with items
- Display integration with recipes
- Show checkoff functionality

### Screenshot 8: Kitchen Equipment
**Caption:** "Personalized to Your Kitchen"
- Show equipment selection screen
- Display various appliances
- Emphasize personalization

## Screenshot Design Tips

1. **Use Real Data:** Show realistic food items, not placeholder text
2. **Consistent Theme:** Use either all light mode or all dark mode
3. **Add Captions:** Overlay short, punchy captions on screenshots
4. **Device Frame:** Consider adding device frames for professional look
5. **Brand Colors:** Use ChefSpAIce green (#27AE60) for accents
6. **Avoid Clutter:** Show clean, organized screens
7. **Show Value:** Each screenshot should highlight a specific benefit

---

## Tools for Creating Screenshots

- **Figma:** Create mockups with device frames
- **Screenshots Pro:** Add device frames and captions
- **AppLaunchpad:** Generate store-ready screenshots
- **Expo:** Run on simulator and use Cmd+S to capture

---

## How to Capture Screenshots in Expo

### iOS Simulator
1. Run `npx expo start` and press `i` for iOS simulator
2. Navigate to the screen you want to capture
3. Press `Cmd + S` to save screenshot
4. Find screenshots in ~/Desktop

### Android Emulator
1. Run `npx expo start` and press `a` for Android emulator
2. Navigate to the screen you want to capture
3. Click the camera icon in emulator toolbar
4. Screenshots saved to emulator's Pictures folder

### Physical Device
1. Use Expo Go app on your device
2. Take screenshots using device controls
3. Transfer to computer for editing
