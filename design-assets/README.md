This folder contains SVG icons and design elements that can be imported into Figma.

1. Open your Figma file
2. Drag and drop SVG files directly onto the canvas

## Customizing in Figma

After importing:
- **Change colors:** Select paths and change stroke/fill colors
- **Resize:** Scale freely (vectors stay sharp)
- **Edit paths:** Double-click to edit individual anchor points
- **Create components:** Convert to Figma components for reuse

## Design Tokens

Recommended colors for ChefSpAIce:
- Primary Green: `#4CAF50`
- Dark Green: `#2E7D32`
- Light Green: `#E8F5E9`
- Gold/AI Accent: `#FFD700`
- Text: `#212121` (dark) / `#FFFFFF` (light)
